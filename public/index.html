Documentation of My Toys API Project

Toys Description Query Type
1 Search products by name : /toys?search=name GET request
2 Search products by info: /toys?search=info GET request
3 Search products by category: /toys?search=category GET request
4 Search oll products: /toys GET request
5 Add products only with TOKEN: /toys POST request
6 Edit products use id products And with TOKEN: /toys/idEdit PUT request
7 delete products use id products And with TOKEN: /toys/idEdit DELETE request
Users Description Query Type
1 get info about user only with TOKEN: /userinfo GET request
2 Add user (make sure to use your email and password only ): /users POST request
3 create user: /users POST request
4 login only with user that sign in: /login POST request
5 delete user only with TOKEN: /idDel DELETE request
